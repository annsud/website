summary: 'Applying independent component analysis to mine spatio-temporal weather data — recovering the North Atlantic Oscillation’s synoptic patterns directly from the data.'
abstract: >-
  In this article we apply the independent component analysis technique for mining
  spatio-temporal data. The technique is applied to mine for patterns in weather
  data using the North Atlantic Oscillation (NAO) as a specific example. We find
  that the strongest independent components match the observed synoptic weather
  patterns corresponding to the NAO. We also validate our results by matching the
  independent component activities with the NAO index.
